##Requirements
Get API Key here

Configure API key in .env.local in root:

GPT_API_KEY=open-ai-api-GPT_API_KEY

First, run the development server:

To run the app
Clone the repo
git clone https://github.com/mouryasumit0/roam-around.git

Install the dependencies
npm install

# or yarn, pnpm, etc...
Run the app
npm run dev
